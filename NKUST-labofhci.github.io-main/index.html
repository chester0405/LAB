關於實驗室

以使用者為中心設計(Human-Centered Design)的方法與應用。

能力培養與要求：撰寫學術性與應用型計畫,撰寫與發表國內外重要會議論文,撰寫英文、碩士論文,建立與業界良好的互動。

人機互動設計

人機互動，是一門研究系統與使用者之間的互動關係的學問。系統可以是各種各樣的機器，也可以是電腦化的系統和軟體。

社群商務

電子商務相關知識，行銷、數據分析、統計分析。

大數據應用

學習大數據3V，學習如何運用巨量資料為企業創造價值。

AI與人機協同合作

人與機器協同合作，滿足少量多樣的生產需求。
